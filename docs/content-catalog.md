# WorkLang AI — Content Catalog

This catalog is the duplicate-prevention inventory and curriculum plan for
lesson authors. Read it together with
[`content-guidelines.md`](content-guidelines.md) before creating content, and
append every newly created lesson in the same change.

Lesson numbers are sequential within each language and career track. The
professional topic summarizes the scenario used across the lesson rather than
merely repeating the career-track name.
